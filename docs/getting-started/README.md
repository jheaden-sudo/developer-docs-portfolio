# Getting Started

## Overview
Improv Workshop API is a REST API that serves structured improv game data for workshop facilitators and training tools.

## Base URL
http://localhost:3000

## Prerequisites
- API key (fictional for this portfolio project)
- Basic REST understanding
- Tool like Postman or curl

## Quick Start

### Request
```bash
curl https://api.improv-workshop.dev/v1/games \
  -H "Authorization: Bearer YOUR_API_KEY"
```
